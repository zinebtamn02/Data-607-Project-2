# DATA 607 – Project 2: Data Tidying

This project demonstrates how wide format datasets can be transformed into tidy datasets using the **tidyr** and **dplyr** packages in R. Three different datasets were imported, cleaned, and reshaped from wide format to long format using `pivot_longer()`. After tidying the data, simple analyses and visualizations were created using **dplyr** and **ggplot2**.

## Datasets

1. CONMEBOL World Cup Qualification Standings 
https://en.wikipedia.org/wiki/2026_FIFA_World_Cup_qualification_(CONMEBOL)

2. NFL Player Statistics Dataset
https://www.kaggle.com/datasets/kendallgillies/nflstatistics

3. Gym Members Exercise Dataset
https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset
